# HR-Analytics-Using-Machine-Learning-and-Clustering
An end-to-end HR Analytics Machine Learning project implementing EDA, PCA, Logistic Regression, Classification Algorithms, ROC-AUC Analysis, and Clustering Techniques for employee behavior and attrition analysis.

##Overview
This project applies Machine Learning and Clustering techniques on HR Analytics data to analyze employee behavior, satisfaction, engagement, and attrition patterns.
The project covers both supervised and unsupervised learning techniques along with dimensionality reduction and model evaluation methods.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Apply Principal Component Analysis (PCA)
- Perform Bias-Variance Analysis
- Build Logistic Regression models
- Compare Classification Algorithms
- Evaluate models using ROC-AUC
- Perform Employee Segmentation using Clustering

## Results Summary
### Best Classification Model
Naive Bayes achieved the best balance between Precision, Recall, and F1 Score for employee attrition prediction.

### Highest Accuracy
Support Vector Machine (SVM)

### Best Clustering Technique
K-Means Clustering
Silhouette Score: 0.1168
